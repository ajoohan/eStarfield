작성일: 2026-07-13
참고 사이트: https://www.realmidas.com

## 1. 개요

하남 미사·유니온시티 권역의 공인중개사무소 **e스타필드 공인중개사사무소**의 브로슈어형 홈페이지.
realmidas.com(마이다스부동산중개법인)의 구조와 신뢰감 있는 코퍼릿 톤을 참고하되, 자체 브랜드/색상으로 제작한다.

- 형태: 정적 브로슈어 + 매물 소개 (데모 데이터)
- 대상: 매물을 찾는 임차인/매수인 및 매물을 내놓는 임대인/매도인
- 목표: 신뢰감 전달 + 문의 전환(전화/문의폼)

## 3. 기술 스택

- **Vite + React (JavaScript)** — 기존 메디프론트 프로젝트와 동일 스택으로 일관성 확보
- **react-router-dom** — 6페이지 라우팅
- **Pretendard** 폰트
- 콘텐츠/매물 데이터: `src/data.js` (단일 소스)
- 배포: Vercel(추후), 현재는 로컬 `D:\estarfield`
- 상태관리/백엔드 없음(문의 폼은 데모, 미연동)

## 5. 사이트 구조

공통 레이아웃: **Header**(로고 + 5메뉴 + 전화 CTA, 모바일 햄버거) / **Footer**(회사정보·주소·약도 링크·연락처).

### 5.1 홈 `/`
- 히어로: 핵심 가치제안(예: "하남 미사·유니온, 딱 맞는 매물을 빠르게") + 전화/문의 CTA
- 매물 유형 4카드: 상가/점포 · 사무실/오피스 · 아파트/주택 · 토지/건물
- 이용방법 3단계: ① 전화·방문 상담 ② 간편 문의 접수 ③ 매물 확인·계약 (realmidas 3방식 참고)
- 대표 매물 미리보기(홈에 3~4개, 매물정보로 연결)
- 하남 지역 강점 섹션(단지 내 위치, 지역 밀착)
- 하단 문의 CTA 배너

### 5.2 회사소개 `/about`
- 대표 인사말, 사무소 소개, 위치(에일린의뜰 단지내 상가 110호), 강점/약속

### 5.3 매물정보 `/listings`
- 유형 필터(전체/상가/사무실/주택/토지) + 거래유형(매매/전세/월세) 필터
- 매물 카드 그리드(`src/data.js` 더미 매물) — 제목, 유형, 거래유형, 면적, 보증금/월세·매매가, 위치, 썸네일
- 카드 클릭 시 상세(모달 또는 상세 섹션): 설명, 세부 스펙, 문의 버튼

### 5.4 계약절차안내 `/process`
- 단계별 타임라인: 매물 확인 → 상담/현장 방문 → 조건 협의 → 계약 체결 → 잔금·이전/입주
- 준비 서류 안내, 유의사항

### 5.5 부동산정책 `/policy`
- 중개보수(수수료) 요율표(매매/임대차, 주택/오피스텔/상가 구분)
- 관련 정책·거래 유의사항, 자주 묻는 질문(FAQ)
- (요율은 일반 안내값 + "실제 적용은 상담" 문구)

### 5.6 문의 `/contact`
- 문의 폼(이름/연락처/문의유형/내용) — **데모, 미연동**(제출 시 안내 메시지)
- 연락처(전화 클릭 통화), 주소, 약도(정적 이미지 또는 지도 링크), 영업시간

## 6. 컴포넌트 구조(초안)

```
src/
  main.jsx
  App.jsx                 // 라우터 + 공통 레이아웃
  data.js                 // 회사정보 + 매물 배열 + 정책/절차 텍스트
  components/
    Header.jsx  Footer.jsx
    Hero.jsx  SectionTitle.jsx
    PropertyTypeCards.jsx  StepGuide.jsx
    ListingCard.jsx  ListingGrid.jsx  ListingFilter.jsx
    CtaBanner.jsx
  pages/
    Home.jsx  About.jsx  Listings.jsx  Process.jsx  Policy.jsx  Contact.jsx
  styles/ (또는 CSS Modules / 단일 CSS + CSS 변수)
```

각 컴포넌트는 단일 책임을 갖고 props로 데이터를 받는다(데이터는 `data.js`에 집중).
